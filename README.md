# 記帳寶 Accounting Baby
一款基於個人需求撰寫的簡易記帳Android APP

## 詳細規格
- API 28，Android 9(Pie)，此為最後一代能用ListView的版本
- 資料庫為APP隨附之SQLite，希望使用者離線也能用得開心～
